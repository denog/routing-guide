# Resources

- [IXP-from-Scratch](https://labs.ripe.net/author/liske/ixp-from-scratch-building-a-new-ix/)

    A report about our endeavor of building DD-IX, an Internet Exchange located in the city of Dresden, Germany.
    DD-IX is driven by a grassroots community that loves contributing to a resilient and efficient Internet.
    We start with some basic background and will share technical design decisions in upcoming articles.

    1. [Part 1: Building a New IX](https://labs.ripe.net/author/liske/ixp-from-scratch-building-a-new-ix/)
    2. [Part 2: The Peering LAN](https://labs.ripe.net/author/liske/ixp-from-scratch-the-peering-lan/)
    3. [Part 3: Network and Security Design](https://labs.ripe.net/author/liske/ixp-from-scratch-network-and-security-design/)

- [MANRS Implementation Guide for Network Operators](https://manrs.org/netops/guide/)

- [IPng Networks: Case Study - BGP Routing Policy](https://ipng.ch/s/articles/2021/11/14/case-study-bgp-routing-policy/)

- [RIPE NCC Academy](https://academy.ripe.net) for e-learning courses regarding e. g. IPv6 and BGP Security.
